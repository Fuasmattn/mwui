# Utility Classes

## Get started

Add the following lines to the index file of your React App. This ensures global styles and utility classes to be present.

```javascript
import { injectGlobalStyles } from "@fuasmattn/mwui";
import "@fuasmattn/mwui/index.css";

injectGlobalStyles();
```

## Spacing

Based on the **spacing tokens**, utility classes are generated.

<div style={{ display: "flex", gap: '24px'}}>
  {Object.values(tokens)
    .filter((t) => t.type && t.type === "spacing" && !t.name.includes("-"))
    .map((t, i) => (
      <div key={i}>
        <p><code>{t.name}</code></p>
        <p><code>{t.value}px</code></p>
        <div style={{ color: 'white', background: tokens.Palette[70].value, width: `${t.value}px`, height: `${t.value}px`}} key={i}>
        </div>
      </div>
    ))}

</div>

### Margin & Padding

export const styleOuter = {
  background: tokens.blue[900].value,
};
export const styleInner = {
  display: "inline-block",
  background: tokens.blue[800].value,
};

Set the margin or padding on one side of an element using the `m{t|r|b|l}-{size}` or `p{t|r|b|l}-{size}` utilities.
Controlling of either vertical or horizontal margins or paddings can be done with the `m{y|x}-{size}` or `p{t|r|b|l}-{size}` utility.
